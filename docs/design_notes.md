# Design Notes

## Problem
Mission-critical systems need structured readiness review, not just raw status reporting.

## MVP question
Are we ready to operate, and if not, what is blocking readiness?

## Human role
The dashboard does not replace the decision-maker. It organizes the evidence a reviewer would use.

## Data included
- requirements
- tests
- anomalies
- risks
- subsystem status
- latest review decision

## Future expansion
- dependency graph
- evidence links
- PostgreSQL backend
- Docker packaging
- more nuanced scoring
